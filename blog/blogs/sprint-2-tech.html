Margin, padding, and borders

To explain the differnces between these differnt CSS propertys we'll first have to imagine a box like this one right here!

And we'll throw in some text inside of it, this will be the content.

The first concept we'll be exploring is padding. When you increase your padding you can imagine the box expanding away from the content inside of it and increases the space between the content and the sides of the box like this.

Next up is the margin. For this we'll need to throw something up above the box.

When you apply margin the box pushes itself away from the other content around it. Rather than padding where the box changes it proportions to get away from the internal contents, margin moves the box away from external contents

Finally theres the border which just... creates a border... but the border also has a few options to customize it such as border width...

Border color...

And border style!

Do note that when increasing the borders width, it almost acts as increasing the margin. While the distance between the edge of the border and the content above doesnt change, but the border extendeds from the box pushing it further down so do be aware of that when using the border property.
